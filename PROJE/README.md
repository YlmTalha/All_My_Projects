# PROJE — Java Okul Teslimleri

Okul dersi kapsamında teslim edilen Java ödev ve proje dosyaları. Dosyalar öğrenci numarası (`230408032`) ön ekiyle adlandırılmıştır.

## Yapı

```
PROJE/School/
├── 230408032_Assignment1.java
├── 230408032_Assignment2.java
└── 230408032_Project.java
```

## İçerik

- **Assignment1 / Assignment2:** Ders kapsamındaki ödev teslimleri.
- **Project:** Dönem projesi teslimi.

## Çalıştırma

```bash
javac 230408032_Project.java
java 230408032_Project
```

> Not: Java sınıf adları rakamla başlayamadığı için dosya içindeki `public class` adı dosya adından farklı olabilir; derlerken üretilen sınıf adını kullanın.
