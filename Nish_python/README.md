# Nish Python — Python Kursu

Python programlama kursu boyunca işlenen ders notları, sınıf projeleri ve haftalık ödevler.

## Yapı

```
Nish_python/
├── Dersler/      # Haftalık ders çalışmaları (Week1 – Week8)
├── Projeler/     # Sınıf içi projeler
│   └── Ekim1/    # HesapMakinesi, VeriTurleri, Operatorler, GirdiCikti
└── Ödevler/      # Haftalık ödevler (her ödev kendi klasöründe)
    ├── 1-2.Hafta/
    ├── 3.Hafta/
    ├── ...
    └── 8.Hafta/
```

## Konular

Kurs, temelden ileri seviyeye doğru ilerler:

- **Hafta 1–2:** Değişkenler, veri tipleri, tip dönüşümleri, operatörler, koşullar (ATM, Hesap Makinesi, BMI, Kalori Hesaplama vb.)
- **Hafta 3:** Listeler ve liste işlemleri (referans/değer kopyalama, eleman ekleme/silme)
- **Hafta 4:** Sözlükler (dict), filtreleme, frekans hesaplama
- **Hafta 5:** Yönetim sistemleri (Banka Hesabı, Kullanıcı, Kütüphane, To-Do List)
- **Hafta 6:** Uygulamalar (Film Arşivi, Telefon Rehberi)
- **Hafta 7:** Oyun ve araçlar (Adam Asmaca, Kelime Sayacı, Dönüşüm Hesaplayıcı)
- **Hafta 8:** Kapsamlı sistemler (Banka, Maaş Yönetimi, Öğrenci Not Sistemi)

## Çalıştırma

```bash
python "Ödevler/7.Hafta/AdamAsmaca/AdamAsmaca.py"
```

> Her ödev klasöründe ilgili `.py` kaynak dosyası bulunur. `.zip` dosyaları teslim arşivleridir.
