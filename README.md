# All My Projects

Bu repo, çeşitli alanlarda yaptığım ders, ödev ve kişisel projeleri tek çatı altında toplar. Her klasör kendi konusuna ait çalışmaları içerir ve içinde detaylı bir `README.md` barındırır.

## İçindekiler

| Klasör | Konu | Dil / Teknoloji |
| --- | --- | --- |
| [`42/`](./42) | 42 School Piscine (Havuz) ve libft alıştırmaları | C, Shell |
| [`Arduino/`](./Arduino) | Arduino potansiyometre ile bilgisayar ses kontrolü | C, Python, Arduino |
| [`Java/`](./Java) | Okul Java alıştırmaları ve LeetCode çözümleri | Java |
| [`Nish_python/`](./Nish_python) | Python kursu dersleri, projeleri ve ödevleri | Python |
| [`PROJE/`](./PROJE) | Okul Java ödev ve proje teslimleri | Java |
| [`Strings and File Handling/`](./Strings%20and%20File%20Handling) | Log dosyası işleme ödevi | Python |
| [`Website_RealEstate/`](./Website_RealEstate) | Emlak (gayrimenkul) tanıtım web sitesi | HTML, CSS, JavaScript |

## Genel Bakış

Bu çalışmalar; C ile düşük seviye programlama, Java ile nesne yönelimli programlama, Python ile algoritma ve veri işleme, ve front-end web geliştirme gibi farklı konuları kapsar. Çoğunlukla eğitim ve öğrenme amaçlıdır.

## Kullanım

İlgilendiğiniz klasöre girip oradaki `README.md` dosyasını okuyarak o projeye özel çalıştırma talimatlarına ulaşabilirsiniz.

```bash
git clone <repo-url>
cd All_My_Projects
```
