# Java — Okul Alıştırmaları & LeetCode

Java diliyle yapılan okul dersleri/alıştırmaları ve LeetCode problem çözümleri.

## Yapı

```
Java/
├── School/      # Ders alıştırmaları
└── LeetCode/    # Algoritma problemi çözümleri (twosum, araba ...)
```

### School
Temel programlamadan nesne yönelimli programlamaya kadar konuları kapsayan alıştırmalar:
- Değişkenler, döngüler ve koşullar (`ders.java`, `derssdongu.java`)
- String metotları (`Stringmetod.java`, `ters.java`)
- Kalıtım / OOP (`kalitim.java`)
- Uygulamalar (`IstanbulkartHesaplama.java`, `diskriminant.java`, `StudentID_Assignment1.java`)
- Sınav ve deneme çalışmaları (`sinav.java`, `deneme.java`, `projedeneme.java`)

### LeetCode
Klasik algoritma soruları (`twosum.java` vb.) için Java çözümleri.

## Çalıştırma

Her dosya bağımsız çalıştırılabilir:

```bash
javac Dosya.java
java Dosya
```

> Not: Bazı dosyalarda public class adı dosya adıyla eşleşmeyebilir; gerekirse sınıf/dosya adını eşitleyerek derleyin.
